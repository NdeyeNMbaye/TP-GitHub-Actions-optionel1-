### Description 
Ce projet est une application backend de type plateforme d’achat en ligne développée avec Spring Boot.
Il permet de gérer les fonctionnalités principales d’un système e-commerce (produits, commandes, utilisateurs).

Le projet est intégré dans une chaîne CI/CD avec Jenkins et conteneurisé avec Docker.

### Pre requis 

Java 17
Spring Boot
Spring Data JPA
PostgreSQL
Maven
Docker
Jenkins (CI/CD)
Git & GitHub
